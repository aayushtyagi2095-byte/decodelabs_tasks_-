# 📊 DecodeLabs SQL Data Analysis – Project 3

## 📌 Project Overview
This project is part of the DecodeLabs SQL Data Analysis portfolio. The objective was to analyze an e-commerce sales dataset using SQL concepts and present business insights through an interactive Excel dashboard.
The project demonstrates data cleaning, SQL query writing, KPI analysis, dashboard creation, and business reporting.

## 🎯 Objectives
- Analyze sales performance using SQL.
- Identify business trends and key performance indicators (KPIs).
- Create a professional Excel dashboard.
- Generate meaningful business insights.
- Practice real-world SQL data analysis techniques.

## 📂 Dataset Information
- **Dataset Type:** E-commerce Sales Data
- **Records:** 1,200+
- **Columns:** 14
- **Tools Used:**
  - Microsoft Excel
  - SQL
  - Pivot Tables
  - Charts & Dashboard
